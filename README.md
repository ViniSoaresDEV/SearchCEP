# SearchCEP

Uma aplicação web simples para consultar CEPs brasileiros usando a API do ViaCEP.

## Funcionalidades

- Busca de endereço por CEP
- Exibe logradouro, bairro, cidade e estado
- Interface responsiva e amigável

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/ViniSoaresDEV/SearchCEP.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd SearchCEP
   ```

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Digite o CEP no campo (8 dígitos, com ou sem hífen).
3. Clique em "Buscar".
4. O resultado aparecerá abaixo.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)

## API

Utiliza a API gratuita do [ViaCEP](https://viacep.com.br/).
